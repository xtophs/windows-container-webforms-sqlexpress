Welcome to the "TheBeerHouse" Starter Kit

This Starter Kit was created as a sample project for the Wrox Press book "ASP.NET 2.0 Website Programming", by Marco Bellinaso.

TheBeerHouse site has everything a modern content-based site will have, including dynamic articles & news, polls for user-to-site communication, forums for user-to-user communication, newsletters and RSS feeds to notify members about new content on the site, an e-commerce store for selling products online, homepage personalization and content localization.

The "ASP.NET 2.0 Website Programming" book follows a “Problem - Design - Solution” approach in each chapter: the Problem section explains the business requirements for the module designed in that chapter, the Design section is used to develop our roadmap for meeting those requirements, and the Solution section is where the code to implement the design is written. At the end of the book the reader will have created a real-world site with most of the features required by modern content-centric sites and e-commerce stores.

If you want to learn everything about this Starter Kit, its design and all the implementation details, please take a look at the book's homepage. You can also contact Marco Bellinaso through his blog, or via e-mail at mbellinaso@dotnet2themax.com.

Running the site

To run the site just press F5 or click Debug | Start Debugging.
The sample administrator account has the following credentials: Username: Admin / Password: admin

Notes

1) You need to add NTFS write permission to the Uploads folder, for the ASPNET (Windows 2000 and XP) or the Network Service user account (Windows Server 2003).
2) Be sure to delete other sample account from the Admin / Manage Users page before going to production.

Further instructions

Please refer to the book's last chapter for detailed instructions about the deployment of the site to a local or remote IIS site, and about how to upgrade from SQL Server 2005 Express Edition to the full-featured SQL Server 2005.
